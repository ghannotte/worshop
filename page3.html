Etudes supérieures

Représentation du contexte

----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Le choix des filières en études supérieures (comparaison avec la Malaisie)

données croisées entre les données de l’éducation nationale française

(rapport annuel global 2019, rapport des écoles d’ingénieurs 2019-2020,

rapport sur les filières de l'enseignement 2019) et Malaisienne(rapport annuel global 2019).

Les valeurs réprésententes le pourcentage de femmes dans chaque filliéres

source: reperes-et-references-statistiques-sur-les-enseignements-2019

Données MESRI-DGESIP-SIES 2019-2020

MALAYSIA EDUCATIONAL STATISTICS. QUICK FACTS 2019

Solutions présentées

----------------------------------------------------------------------------------------------------------------------------------------------------------------------

- Conclure à partir du modèle de réussite de la Malaisie où, par exemple,

le nombre d'enseignantes dans le domaine du numérique est plus élevé qu’en France.

- L’idée serait également de mieux communiquer sur les données relatives à l’inclusion des femmes dans le numérique en France.

Cela passe par la création d’une base de donnée ouverte disponible sur les sites officiels des écoles supérieurs

dans le domaine du numérique pour être transparent sur les quotas de parité femme / homme dans chaque spécification.

- Accentuer la présence des métiers techniques du numérique représentés par des femmes dans les forums de formation et des métiers

Définition du problème

----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Dans le cadre de la parité femme / homme dans les métiers techniques du numérique,

le modèle français est moins performant que le modèle malaisien.
